# Cat Facts

## _How to run_

- `npm i`
- `npm run dev`
- Go to `localhost:3000`
- ✨Magic ✨

## _How to lint_

`npm run tsc` - run ts check

`npm run lint` - run eslint check

`npm run lint -- --fix` - run eslint and fix errors

`npm test` - run tests

## _How to build_

`npm run build` - Build app for production
